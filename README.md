# BG — AI & Computer Vision Consulting

A minimalist, Notion-style single-page landing page for professional AI auditing and computer vision consulting services. 

## Live Site
**[https://zosob.github.io/consulting-page/]**

## 🛠️ Tech Stack
- **Frontend:** Semantic HTML5, CSS Grid/Flexbox
- **Theme Engine:** Vanilla JS with `localStorage` persistence for instant Light/Dark mode switching
- **Typography:** DM Sans & Playfair Display (Google Fonts)
- **Deployment:** Automated hosting via GitHub Pages
